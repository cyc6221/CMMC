## Notation

- If $n$ is a positive integer, then $\mathbb{Z}_n$ denotes the set $\{0, \ldots, n-1 \}$, and $[n]$ or $[1..n]$ denote the set $\{1, \ldots, n \}$.

- Denote the number of coordinates of a vector $x$ by $\lvert x\rvert$. If $x$ is a vector, then
  - $\lvert x\rvert$ denotes its length, the number of coordinates of $x$,
  - $x[i]$ denotes the $i$-th coordinate of $x$
  - $[x] = \{x[i]: 1 \le i \le \lvert x\rvert\}$ denotes the set of all coordinates

- A string is identified with a vector over $\{ 0, 1 \}$. If $x$ is a string, then
  - $x[i]$ denotes its $i$-th bit
  - $\lvert x\rvert$ denotes its length.

以 $\varepsilon$ 表示空向量或空字串。集合 $S$ 的大小記為 $\lvert S\rvert$。對於集合 $D,R$，以 $\mathrm{FNS}(D,R)$ 表示所有函數 $f:D\to R$ 的集合。

### Sampling and algorithm notation

設 $S$ 為有限集合。記

$$
x \xleftarrow{$} S
$$

表示從 $S$ 中均勻隨機抽取一個元素並指定給 $x$。

記

$$
y \leftarrow A^{O_1,\ldots}(x_1,\ldots; r)
$$

表示執行演算法 $A$，其輸入為 $x_1,\ldots$，隨機幣為 $r$，並可存取 oracle $O_1,\ldots$，最後令 $y$ 為其輸出結果。

記

$$
y \xleftarrow{$} A^{O_1,\ldots}(x_1,\ldots)
$$

表示先隨機選取 $r$，再令

$$
y \leftarrow A^{O_1,\ldots}(x_1,\ldots; r).
$$

記

$$
[A^{O_1,\ldots}(x_1,\ldots)]
$$

表示當 $A$ 以輸入 $x_1,\ldots$ 並可存取 oracle $O_1,\ldots$ 執行時，所有可能輸出的集合。

除非另有說明，演算法皆為 randomized。執行時間以 worst case 計。

## Games

我們使用文獻 [11] 的 code-based game playing framework（範例見 Fig. 3）。

一個 game 具有若干 procedures，也稱為 oracles。其中包括 `Init` 與 `Fin`。在以 game $\mathrm{Gm}$ 執行 adversary $\mathcal{A}$ 時，首先執行 procedure `Init`，其回傳值作為 $\mathcal{A}$ 的輸入。之後，$\mathcal{A}$ 可以呼叫除了 `Init`、`Fin` 之外的所有 game procedures。

當 adversary 終止時，其輸出被視為 `Fin` 的輸入，而 `Fin` 的回傳值即為 game 的輸出。

以

$$
\Pr[\mathrm{Gm}(\mathcal{A})]
$$

表示 game $\mathrm{Gm}$ 與 adversary $\mathcal{A}$ 一起執行後輸出為 `true` 的事件。

在撰寫 game 或 adversary 的 pseudocode 時，預設 boolean 變數初始化為 `false`，整數變數初始化為 $0$，而 set-valued 變數初始化為空集合 $\emptyset$。

當 adversary $\mathcal{A}$ 與 game $\mathrm{Gm}$ 一起執行時，adversary 的執行時間記為 $T_{\mathcal{A}}$，並假設 game procedures 的回應時間皆為單位時間。

以

$$
Q_{\mathcal{A}}^O
$$

表示在該次執行中，$\mathcal{A}$ 對 oracle $O$ 所做的查詢次數。這些計數皆以 worst case 計。

## Groups

設 $G$ 為一個階為 $p$ 的群。我們對群運算採用乘法記號，並以 $1_G$ 表示 $G$ 的單位元素。

記

$$
G^* = G \setminus {1_G}
$$

表示所有非單位元素的集合；若 $G$ 的階為質數，則這也是 $G$ 的所有生成元所成的集合。

若 $g\in G^*$ 是一個生成元，且 $X\in G$，則 $X$ 以底數 $g$ 的離散對數記為

$$
\mathrm{DL}*{G,g}(X),
$$

且它屬於集合 $\mathbb{Z}*{\lvert G\rvert}$。
